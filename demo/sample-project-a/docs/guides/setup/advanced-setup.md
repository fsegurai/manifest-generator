# Advanced Setup

This guide covers advanced setup options for your project.

## Environment Variables

Set these environment variables for production:

- `NODE_ENV
- `API_KEY=your-api-key`
- `DATABASE_URL=your-database-url`
- `CACHE_ENABLED=true`
- `LOG_LEVEL=info`
- `FEATURE_FLAG_X=true`
- `FEATURE_FLAG_Y=false`

---

## Configuration Files

Ensure your configuration files are set up correctly:

- `config/appConfig.js`
- `config/databaseConfig.js`
- `config/cacheConfig.js`
- `config/loggingConfig.js`
