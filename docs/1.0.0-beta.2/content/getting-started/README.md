# Getting Started

This section is for external users who want to install and use `@fsegurai/manifest-generator` quickly.

## Success checklist

- You can run the CLI (`--help` or `--version`)
- You can generate `manifest.json` and `search-index.json`
- You understand which command to use for one route vs many projects

## Fastest path

```bash
# Run without installing globally
npx @fsegurai/manifest-generator --help
npx @fsegurai/manifest-generator --route ./docs
```
